About This Project

This project was created as part of my learning journey in web development. The goal was to build a simple but functional web application using only HTML, CSS, and JavaScript.

The website uses a public API to fetch random cat facts. This helped me understand how to work with APIs, handle asynchronous JavaScript using async/await, and update the DOM dynamically.

While working on this project, I also practiced responsive design to ensure the website works well on different screen sizes. I focused on keeping the design clean and user-friendly, using a simple color scheme and readable fonts.

Another important part of this project was learning how to debug errors using browser developer tools and writing basic tests to check if my functions work correctly.

Overall, this project helped me improve my understanding of frontend development and gave me more confidence in building interactive web pages without using frameworks.
